import 'http' from index;
import 'system' from index;

@getter('http://:localhost:8000/')
class host extends server {
  func host ('http') = {
      system.host(port:8000);
  }
}

@setter('/app/hello_world!') = {
  system.println('hello_world!');
  system.call(host);
}

Clean error handling
No Runtime Exceptions. Clove uses type inference to detect corner cases and give friendly hints. SarkarLab switched to Clove about one year ago, and 25k+ lines later, they still have not had to scramble to fix a confusing runtime exception in production.

High performance
Clove is stable. And thanks to its statically typed core, we can compete right now with other full stack languages such as Dart, Elm or Ruby.

Moreover it is built on a cross-target interpeter, which means it can compile to multiple languages such as C, JavaScript or C++.
